# 🔪 sMurder

> A modern and highly customizable Murder Mystery plugin for Minecraft 1.8.8, featuring multiple game modes, cosmetics, companions, mystery boxes, private rooms and an Among Us inspired experience.

---

## ✨ Features

### 🎮 Multiple Game Modes

- Classic Murder
- 🔥 AMONG US Mode (Sabotage)
- TNT Tag
- All Murderers
- All Detectives
- Hide & Seek

Every mode has its own gameplay logic.

---

# 🚪 Private Rooms

Create your own room.

Features:

- Custom room name
- Colored room names
- Password protected rooms
- Public / Private
- Custom player limit
- Select game mode
- Start game manually
- Select Murder
- Select Detective
- Map voting

---
